# AI Media Lab Website v1.0

화이트 & 블랙 기반의 초기 운영 대시보드입니다.

## 실행 방법
1. 압축을 해제합니다.
2. `index.html` 파일을 더블클릭합니다.
3. 브라우저에서 사이트가 열리면 정상입니다.

## GitHub Pages 배포
저장소에 전체 파일을 업로드한 뒤 Settings > Pages에서 배포합니다.

## 구조
- `index.html` : 메인 화면
- `assets/css/style.css` : 디자인
- `assets/js/main.js` : 체크박스, 진행률, 빠른 이동 기능
